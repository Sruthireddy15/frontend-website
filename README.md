# EduTech Academy

## Educational Website Project

This is a responsive educational website developed as part of the MANTRA 2026 Website Development Assignment.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Responsive Design (Mobile, Tablet & Desktop)
- SEO-Friendly Web Pages
- Home, About, Courses, Gallery & Contact Pages
- Student Registration Form with JavaScript Validation
- Course Information Cards
- Image Gallery
- Easy Navigation Menu
- Clean and Modern User Interface
